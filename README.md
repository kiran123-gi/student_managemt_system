# 📚 Student Management System (CRUD + Search) - Django + MySQL

## 🚀 Project Overview

This project is a full-stack web application built using Django and MySQL that performs complete CRUD (Create, Read, Update, Delete) operations on student data.

Users can add, view, update, delete, and search student records through a simple and user-friendly interface.

---

## 🛠️ Tech Stack

* Python
* Django
* MySQL
* HTML

---

## ✨ Features

* Add new student details
* View all students in a table format
* Update existing student records
* Delete student data
* Search students by name or roll number

---

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/kiran123-gi/Student_management-django-mysql-.git
```

2. Navigate to project folder:

```
cd Student_management-django-mysql-
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Configure MySQL database in `settings.py`

5. Run migrations:

```
python manage.py makemigrations
python manage.py migrate
```

6. Start the server:

```
python manage.py runserver
```
